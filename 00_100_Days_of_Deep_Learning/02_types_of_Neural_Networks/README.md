|----------------------------------|------------------------------------------------------------------------|
| **Perceptron**                   | Simplest neural network unit used for binary classification tasks.     |
| **Convolutional Neural Networks**| Specialized for image and video recognition by extracting spatial features.|
| **Recurrent Neural Networks**    | Designed for sequential data like text or time series, maintaining memory.|
| **Long Short-Term Memory (LSTM)**| An advanced RNN variant that solves long-term dependency problems.     |
| **Backpropagation**              | Algorithm used for training neural networks by minimizing error.       |
